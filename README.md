# iforgot

A CLI tool that helps you fetch the commands that you always forget. 

Installing iforgot
```
$ sudo curl -L -o /usr/local/bin/iforgot https://git.io/vxZYX && sudo chmod +x /usr/local/bin/iforgot
```
